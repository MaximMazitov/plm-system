## 🎨 Как использовать цвета в коде:

### Кнопки
```tsx
// Основная кнопка (фуксия)
<button className="bg-primary-600 hover:bg-primary-700 text-white">
  Сохранить
</button>

// Серая кнопка
<button className="bg-gray-200 hover:bg-gray-300 text-gray-900">
  Отмена
</button>
```

### Бейджи статусов
```tsx
// Используют корпоративные цвета
<Badge variant="success">Утверждено</Badge>  // зеленый
<Badge variant="warning">На проверке</Badge>  // желтый
<Badge variant="danger">Отклонено</Badge>    // красный
<Badge variant="info">Черновик</Badge>       // синий
```

### Фон
```tsx
// Фон страницы - мягкий серый
className="bg-gray-50"

// Фон карточек - белый
className="bg-white"

// Градиент на Login - от фуксии
className="bg-gradient-to-br from-primary-50 to-primary-100"
```
